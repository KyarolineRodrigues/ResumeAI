# ResumeAI

AI-powered Resume Screening and Job Matching Platform.

## Goal

ResumeAI helps candidates compare their resumes with job descriptions by:

* Extracting text from resumes
* Identifying skills
* Matching skills with job requirements
* Showing missing skills
* Providing recommendations

## Technologies (Planned)

* Python
* FastAPI
* NLP
* Machine Learning
* React
* GitHub
